3D VR Chemical Training Software

Screenshot of the 3D Software

This 3D VR software consists of various stations where you will have to perform certain tasks in a chemical lab. Each station includes a checklist of steps that must be completed to accomplish the objectives. The software is designed to provide a realistic and immersive training experience, helping users to practice and master essential lab procedures safely.

  • Created a login system using text files(.JSON, .txt and .cs) as a database and integrate it on VR project
  • Implement trigger events as to the certain markers for the workers to practice on a task
  • Powered by the Unity Engine and Steam VR API.
  • Developed a heatmap project that display how frequent a user stares at or how long the user stand idle. Powered by the Unity Engine and Steam VR API.
