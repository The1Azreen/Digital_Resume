ONLINE CARD GAME AVIATORS IN ACTION

Screenshot of the game

This online card game is designed to educate children about the aviation industry. Players will learn about different types of aircraft, aviation terminology, and the roles of various aviation professionals. The game combines fun and learning by incorporating engaging gameplay mechanics with educational content, making it an enjoyable experience for kids while providing valuable knowledge about the world of aviation.

  • The game is powered by the Unity Engine 2019 for the game logic
  • Device uses php for server side info (e.g player turn and deck data)
